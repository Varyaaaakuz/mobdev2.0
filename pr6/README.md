Создали новый проект Android с именем FragmentApp и начали с настройки зависимостей в файле build.gradle, где добавили библиотеку androidx.fragment для работы с фрагментами. В главной активности MainActivity реализовали проверку состояния savedInstanceState через условие if - это обеспечивало, что фрагмент добавляется только при первоначальном запуске приложения, а не при каждом его пересоздании. Для передачи данных использовали объект Bundle, в который поместили номер студента из списка и установили этот Bundle в качестве аргументов фрагмента. Затем создали фрагмент StudentFragment, который в методе onViewCreated с помощью requireArguments() извлекал переданный номер и отображал его в текстовом поле TextView. Весь этот процесс позволил на практике освоить базовые принципы работы с FragmentManager и механизм передачи данных между активностью и фрагментом.
